About the Formula 1 API

The Formula 1 API is a RESTful API which means that it is stateless and uses HTTP requests to GET, PUT, POST and DELETE data. The API is documented using Swagger UI which is a tool for visualizing and interacting with the API's resources. Follow this link to get to the API.


Getting Started

To use the API, follow these simple steps:

  • Include the Endpoints in your service file.
  • Write the HTTP-requests in your backend controller.
  • Call the HTTP-requests in your service file.


    • Remember to change the port when including the URL:

  • http://localhost:XXXX/api/drivers
  • http://localhost:XXXX/api/imageUpload
  • http://localhost:XXXX/teams // Under development
  • http://localhost:XXXX/races // Under development


    • HTTP-requests

  • 'GET /api/drivers': Retrieve all drivers
  • 'POST /api/drivers': Add new drivers to the race
  • 'PUT /api/drivers': Update/change the information of a driver
  • 'GET /api/drivers/{id}': Search for a driver by ID
  • 'GET api/drivers/GetByName/{name}': Search for a driver by name
  • 'DELETE' api/drivers/{id}: Delete a driver by ID


    • Data

  • ID (int)
  • Name (string)
  • Age (int)
  • Nationality (string)
  • Image (string)
